
## Structure

```txt
backend/   ExpressJS REST API
frontend/  ReactJS UI with Vite
```

## Setup

Install dependencies from the project root:

```bash
npm install
npm run install:all
```

Run both backend and frontend:

```bash
npm run dev
```

Backend runs on `http://localhost:9999`.
Frontend runs on `http://localhost:5173`.
